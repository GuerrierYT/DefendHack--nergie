# WattLab

WattLab est un site de vulgarisation scientifique sur l'energie, cree pendant le Defend Hack 2 organise par Defend Intelligence.
Le projet transforme des notions parfois abstraites en mini-jeux interactifs, courts et visuels.

## Concept

L'objectif de WattLab est d'aider a comprendre l'energie par la manipulation.
Plutot que de lire uniquement des definitions, l'utilisateur teste, compare et observe des ordres de grandeur.

Le ton du site est volontairement neutre, rigoureux et ludique : chaque mini-jeu isole une idee scientifique et la rend plus concrete.

## Mini-jeux

- **Atelier propulsion** : construire une voiture en choisissant une source d'energie, un stockage embarque, une motorisation et une transmission.
- **Mesure 1 kWh** : composer une consommation pour comprendre ce que represente 1 kWh.
- **Suis les pertes** : observer les pertes a chaque conversion d'energie.
- **Passe la nuit** : comparer plusieurs solutions de stockage pour couvrir une demande du soir.
- **Compose le mix** : construire un mix electrique en equilibrant CO2, cout et stabilite.
- **Badges WattLab** : suivre les observations reussies et les pistes restantes a explorer.

## Fonctionnalites

- Interface web statique, jouable directement dans le navigateur.
- Mini-jeux interactifs avec scores, badges et retours pedagogiques.
- Visualisations simples : jauges, flux de billes, graphiques, mix electrique, chaine de propulsion.
- Sauvegarde des badges pendant la session de navigation.
- Garage local pour conserver les voitures testees sur le navigateur.

## Lancer le projet

Le projet ne necessite pas d'installation.

Ouvrir simplement le fichier `index.html` dans un navigateur.

```text
index.html
```
